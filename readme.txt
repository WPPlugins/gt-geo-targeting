== Description ==

Geo-Targeting (GEO target) content based on user’s country.

Features:

* Shortcodes for use in content.
* Show the content to users in multiple countries.
* Hide the content from users in multiple countries.
* Show a note to users who are not able to see the content.
* PHP Template tags for use it templates.

More features will be added in future versions.

== Installation ==

1. Unzip gt-geotargeting.zip in 'wp-content/plugins'.
1. Activate the plugin through the 'Plugins' menu in WordPress
